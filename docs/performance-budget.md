# 性能预算

> 按 R-10.2.1：任一指标回归 >10% MUST 在 PR 说明原因。
> 按 R-10.2.2：CI MUST 集成 size-limit，超预算直接失败。

---

## 启动指标

| 指标 | 预算 | 测量环境 |
|---|---|---|
| 冷启动至工作台首帧 | ≤ 3000ms | Windows 11, 16GB RAM, SSD |
| 热启动（已有 workbench 缓存）| ≤ 1500ms | 同上 |
| 工作台初次布局完成 | ≤ 5000ms | 同上 |

---

## 交互响应

| 指标 | 预算 |
|---|---|
| 路由切换（当前无路由，预留） | ≤ 300ms |
| Tab 切换（含光标/滚动恢复）| ≤ 100ms |
| 主题切换（单帧内） | ≤ 16ms |
| ShellCheck 分析响应（防抖后）| ≤ 1500ms |
| 格式化(shfmt)响应 | ≤ 500ms |
| 列表渲染（≤5000 文件）| ≤ 200ms |
