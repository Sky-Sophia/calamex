区域级过渡规范

每条写死「用什么属性 / 多长 / 什么曲线」。
场景 时长 · 曲线 属性与行为
侧栏 hover 80ms · ea-o 仅 bg + color。不位移、不缩放
侧栏 active 切换 180ms · ea 2px pill 用 translateY 滑动；背景 120ms 淡入，同时起
编辑器 tab 切换 120ms · ea-io 顶线横滑；内容 60ms fadeOut → 60ms fadeIn
Terminal 面板 240ms · ea height 0→180px，内容 80ms 延迟淡入
Drawer 240ms · ea translateX，backdrop 120ms 淡入；退出反序
Dialog 320ms · ea Backdrop 160ms；对话框 opacity + translateY 4px，延迟 40ms
Dropdown / Command 120ms · ea-o opacity + translateY -3px；关闭 80ms 原地淡出
Tooltip 400ms 延迟 · 80ms hover 延迟 400ms 才显现；移开立即消失无淡出
Toast 180ms · ea-o translateY 16px 滑入 + opacity；纯缓出、无 overshoot

绝对禁忌

✗ hover 时元素缩放（transform:scale）— Linear 从不用
✗ 弹跳曲线用于大面积移动 — overshoot 只用于 ≤20px 微动
✗ 页面切换用滑动 — 靠内容区淡入即可
✗ 单个过渡 > 320ms — 任何单属性超过都「太显眼」
✗ 动画串联等待 — B 必须在 A 结束前 40~80ms 起步
✗ hover 延迟 > 100ms — 必须即时
✗ 装饰性动画 — 每个过渡都要服务于状态认知
